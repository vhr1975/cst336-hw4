What is Deep Learning?


Deep learning is a part of machine learning based on neural networks. The Learning can be supervised, semi-supervised or unsupervised. Deep learning use deep neural networks, deep belief networks, recurrent neural networks and convolutional neural networks. These advances have been applied to computer vision, speech recognition, natural language processing, audio recognition, social network filtering, machine translation, bioinformatics, drug design, medical image analysis, material inspection and board game programs and many more. The results produed have been comparable to and in some cases superior to human experts. Machine Learning is composed of all the following Deep Learning Types, Deep Learning Supervised, and Deep Leasnring Unsupervised.
01 / Spervised Deep Learning 02 / Unsupervised Deep Learning
Supervised Learning is when the features are extracted manually. Unsupervised Leanring is when the deep learning network extracts all the necessary features automatically.
