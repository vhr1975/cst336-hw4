What is Machine Learning?


Machine Learning is composed of all the following Learning Types, Learning Vision, and Learning Technology.


01 / Learning Types 02 / Learning Vision 03 / Learning Technology
Machine Learning can consist of two types of learning: Supervised and Unsupervised. Supervised learning consists of adding specific features for the algorithm. The features will help the algorithm to derive meaning and context. Unsupervised learning does not provide features and allows the algorithm to learn the meaning and context on its own. Machine Vision is image recognition, object detection and tracking, automatic document analysis, face detection and recognition, computational photography, augmented reality,, 3D reconstruction Machine Technology is vast and rich. There are the cloud vendors AWS, Azure and Google. There is a wide open sosource tools set TensorFlow, Scikit-learn and python.
